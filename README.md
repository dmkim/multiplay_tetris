# 네트워크 테트리스 (Multiplayer Tetris)

## 프로젝트 소개

이 프로젝트는 웹 기반의 2인용 네트워크 테트리스 게임입니다. 사용자는 방을 생성하거나 기존 방에 참여하여 다른 플레이어와 실시간으로 테트리스 대결을 펼칠 수 있습니다.

## 주요 기능

-   **방 생성 및 참여**: 방장이 방을 생성하고, 생성된 방 코드를 통해 다른 플레이어가 방에 참여할 수 있습니다.
-   **게임 준비 시스템**: 각 플레이어는 게임 시작 전 '준비' 상태를 설정할 수 있으며, 두 플레이어 모두 준비되면 방장이 게임을 시작할 수 있습니다.
-   **실시간 게임 동기화**: Socket.IO를 사용하여 두 플레이어 간의 게임 상태(블록 위치, 점수, 레벨 등)를 실시간으로 동기화합니다.
-   **WASD 조작**: 각 플레이어는 WASD 키를 사용하여 자신의 테트리스 블록을 조작합니다.
-   **승패 판정**: 한 플레이어가 게임 오버되면 다른 플레이어가 승리합니다.

## 기술 스택

-   **Frontend**: HTML, CSS, JavaScript
-   **Backend**: Node.js, Express.js
-   **Real-time Communication**: Socket.IO

## 시작하기

### 1. 프로젝트 클론

```bash
git clone <repository_url>
cd <project_directory>
```

### 2. 의존성 설치

프로젝트 루트 디렉토리에서 다음 명령어를 실행하여 필요한 Node.js 패키지를 설치합니다.

```bash
npm install
```

### 3. 서버 실행

다음 명령어를 실행하여 게임 서버를 시작합니다.

```bash
node server.js
```

서버가 성공적으로 실행되면 터미널에 `Server running on port 3000` 메시지가 표시됩니다.

### 4. 게임 접속

웹 브라우저를 열고 다음 주소로 접속합니다:

```
http://localhost:3000
```

## 게임 방법

1.  **방 만들기**: 한 플레이어가 '방 만들기' 버튼을 클릭하여 새로운 방을 생성합니다. 화면에 표시되는 방 코드를 다른 플레이어에게 알려줍니다.
2.  **방 참여하기**: 다른 플레이어는 방 코드를 입력란에 입력하고 '참여하기' 버튼을 클릭하여 해당 방에 접속합니다.
3.  **게임 준비**: 두 플레이어 모두 로비 화면으로 전환되면, 각자 '게임 준비' 버튼을 클릭합니다.
4.  **게임 시작**: 두 플레이어 모두 준비 상태가 되면, 방장의 화면에서 '게임 시작' 버튼이 활성화됩니다. 방장이 이 버튼을 클릭하면 게임이 시작됩니다.
5.  **게임 플레이**: 각 플레이어는 WASD 키를 사용하여 자신의 테트리스 블록을 조작합니다.
    -   **W**: 블록 회전
    -   **A**: 왼쪽 이동
    -   **S**: 아래로 빠르게 이동 (하드 드롭)
    -   **D**: 오른쪽 이동
6.  **승패**: 한 플레이어의 블록이 화면 상단에 닿아 더 이상 내려올 수 없게 되면 게임 오버가 되며, 상대방이 승리합니다.
